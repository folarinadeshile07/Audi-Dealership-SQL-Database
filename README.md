# Audi-Dealership-SQL-Database

## Overview

This project is a relational SQL database for an Audi car dealership. Shows core SQL skills used in database management, and data analysis.

## Features

- Relational Database design
- SQL joins
- Views
- Functions
- Primary and Foreign Keys
- Trigger to automatically update vehicle status after sale
- Aggregate Queries

## Database Tables

- Vehicles
- Customers
- Sales
- Salesperson
- Payment

  ## Files

  - Audidb_DDL.sql - Makes the database schema
  - Audi_DML.sql - Adds in sample dealership data
  - AudiQueries.sql - Example SQL queries used to retrieve and analyze data
